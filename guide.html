Guide

Step 1: Invite HaiBot.

If you haven't already, you can invite HaiBot using this link. Basically all you have to do in this step is click the link, choose the permissions (the ones selected are recommended for full functionality of HaiBot) then click Authorize.

Invite HaiBot!

Step 2: Setup command.

If you're the owner of the server (and you have to be to run this command), HaiBot should've DMed you with pretty much the same info as here. To setup roles for HaiBot, type *setup.

Step 3: HaiBot Mod role.

If you want to use moderation features, you will need to have the HaiBot Mod role. HaiBot will have created this when you ran the *setup command. Also, give this role to all your staff members, as it allows them to use the HaiBot moderation features.

You're done!

That wasn't hard, was it? If you couldn't be bothered to read all of that, we have a video tutorial here. Please note that it may be out of date.
